# Hotel Management System

A Java application for hotel management with role-based access (Manager and Receptionist).  
This project implements multiple software design patterns and provides a clean GUI for hotel operations.

## Features

### Manager
- Secure login and password recovery
- Worker management (add/edit/delete)
- View worker and resident information
- Track hotel income (weekly, monthly, annual)
- Monitor room status

### Receptionist
- Secure login
- Resident management (add/edit/delete)
- Calculate resident costs
- Room assignment and availability tracking
